## วัตถุประสงค์ของหน้านี้

หน้านี้เป็นช่องทางให้ผู้ใช้ **แจ้งข้อมูลที่อาจต้องปรับปรุง** หรือ **เสนอแหล่งข้อมูลทางการ** เพื่อช่วยพัฒนาคลังกฎหมายการศึกษา

เว็บไซต์นี้จัดทำเพื่อ **การศึกษาและค้นคว้าเบื้องต้น** ไม่ใช่ช่องทางให้คำปรึกษาทางกฎหมายเฉพาะกรณีหรือข้อพิพาทส่วนบุคคน

ข้อเสนอแนะทุกรายการจะได้รับการ **พิจารณาก่อน** นำไปแก้ไขเนื้อหา — ไม่รับประกันว่าจะแก้ไขหรือตอบกลับทุกเรื่อง

## แจ้งข้อมูลผิดพลาดหรือข้อมูลล้าสมัย

หากพบว่าสรุปอาจทำให้เข้าใจคลาดเคลื่อน ชื่อกฎหมายไม่ตรงแหล่งทางการ หรือสถานะกฎหมายเปลี่ยนไป กรุณาแจ้งพร้อม **แหล่งทางการ** ที่อ้างอิงได้

## แจ้งลิงก์แหล่งข้อมูลทางการที่เปิดไม่ได้

หากปุ่ม “เปิดแหล่งข้อมูลทางการ” หรือลิงก์ในรายการอ้างอิงใช้งานไม่ได้ กรุณาระบุ URL ที่เสียและ URL ทางการที่ควรใช้แทน (ถ้าทราบ)

## เสนอแหล่งข้อมูลหรือกฎหมายที่ควรเพิ่มเติม

เสนอพระราชบัญญัติ ระเบียบ ประกาศ หรือเอกสารทางการที่เกี่ยวกับการศึกษา — โปรดระบุชื่อเรื่องและลิงก์แหล่งทางการ การเสนอไม่เท่ากับการรับเข้าคลังทันที

## เสนอแนะการปรับปรุงเว็บไซต์

ข้อเสนอเกี่ยวกับการค้นหา แบบทดสอบ เส้นทางการเรียนรู้ หรือการใช้งานบนมือถือ

## สิ่งที่ควรแนบมาด้วยเมื่อแจ้งข้อมูล

- ชื่อหน้าหรือ URL ที่เกี่ยวข้อง
- รายละเอียดที่คิดว่าควรแก้ไข (ชัดและกระชับ)
- ลิงก์แหล่งทางการ (ราชกิจจานุเบกษา กฤษฎีกา หน่วยงานรัฐ) ถ้ามี
- วันที่ตรวจพบปัญหา (ถ้าสะดวก)

## ข้อจำกัดของการตอบกลับและการแก้ไข

- ทีมดูแลจะพิจารณาตามลำดับความสำคัญและแหล่งที่ยืนยันได้
- ไม่รับประกันเวลาตอบกลับหรือการแก้ไขทุกเรื่อง
- ไม่ให้คำปรึกษาทางกฎหมายเฉพาะกรณีผ่านช่องทางนี้
- ก่อนใช้อ้างอิงอย่างเป็นทางการ ควรตรวจสอบตัวบทฉบับทางการด้วยตนเอง

## ความเป็นส่วนตัวเบื้องต้น

กรุณาหลีกเลี่ยงการส่งข้อมูลส่วนบุคคลที่ไม่จำเป็น เช่น เลขประจำตัวประชาชน ข้อมูลสุขภาพ ข้อมูลเด็ก หรือรายละเอียดคดี/ข้อพิพาทส่วนบุคคล

การส่งผ่าน **GitHub Issues** จะเป็นข้อมูลสาธารณะบน GitHub — อย่าใส่ข้อมูลที่ไม่ต้องการเผยแพร่

## ช่องทางแจ้งข้อมูล

เลือกช่องทางด้านล่างตามความสะดวก (รายละเอียดอยู่ในกล่อง “ช่องทางแจ้งข้อมูล” ใต้เนื้อหานี้)
