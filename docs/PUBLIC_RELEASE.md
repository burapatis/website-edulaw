# การเปิดใช้งานสาธารณะ (Public Release)

เอกสารนี้อธิบายการปล่อย **Public Beta** ของ **Education Law Hub** (เว็บไซต์กฎหมายการศึกษา)

## วัตถุประสงค์ของ Public Beta

เปิดให้ผู้ใช้ทั่วไปเข้าถึงคลังความรู้กฎหมายการศึกษาไทยในรูปแบบที่จัดระบบ ค้นหาได้ และมีเครื่องมือเรียนรู้เบื้องต้น — ขณะยอมรับว่าเนื้อหายังต้องขยายและทบทวนอย่างต่อเนื่อง

สถานะปัจจุบัน: **Public Beta** · เวอร์ชัน **1.0.0-beta** (ดู `hugo.toml` → `params.version`)

## สิ่งที่เว็บไซต์ให้ในปัจจุบัน

| ความสามารถ | คำอธิบายสั้น |
|------------|----------------|
| คลังกฎหมาย | สรุปกฎหมายการศึกษาที่คัดสรร พร้อม metadata และลิงก์แหล่งทางการ |
| ค้นหา | Pagefind ค้นหาในเนื้อหาเว็บไซต์ |
| Taxonomy / ตัวกรอง | สำรวจตามหมวดหมู่ หน่วยงาน กลุ่มผู้ใช้ |
| แบบทดสอบ | ทบทวนความเข้าใจเบื้องต้น |
| เส้นทางการเรียนรู้ | ลำดับการอ่านตามบทบาท |
| คู่มือ | บทความอธิบายภาษาเข้าใจง่าย |
| ภาพรวมเว็บไซต์ | สถิติและสถานะ metadata เบื้องต้น |
| แจ้งข้อผิดพลาด | ช่องทาง feedback และ GitHub Issues |
| นโยบาย | แหล่งข้อมูล การอัปเดต ความน่าเชื่อถือ |

## สิ่งที่เว็บไซต์ไม่ให้

- **ไม่ใช่** เว็บไซต์ราชการหรือราชกิจจานุเบกษา
- **ไม่ใช่** บริการให้คำปรึกษาทางกฎหมายเฉพาะกรณี
- **ไม่รับประกัน** ว่าคลังกฎหมายครบทุกฉบับในประเทศ
- **ไม่รับประกัน** ว่าทุกหน้าเป็นฉบับล่าสุดหรือถูกต้องครบถ้วน
- **ไม่แทน** การอ่านตัวบทกฎหมายฉบับเต็มจากแหล่งทางการ

## วิธีใช้ข้อมูลอย่างปลอดภัย

1. อ่านสรุปเพื่อ **เข้าใจภาพรวม**
2. เปิด **แหล่งทางการ** จากหน้ากฎหมาย (`source_url`)
3. ตรวจ **วันที่และสถานะ** กฎหมายในราชกิจจานุเบกษาหรือหน่วยงานที่เกี่ยวข้อง
4. ใช้ [คู่มือ: วิธีใช้เว็บไซต์อย่างปลอดภัย](/guides/how-to-use-edulaw-safely/)
5. หากใช้ในงานทางการหรือข้อพิพาท — ปรึกษาผู้เชี่ยวชาญ

## สิ่งที่ควรตรวจจากแหล่งทางการ

- ชื่อเรื่อง ปี พ.ศ. และฉบับแก้ไขเพิ่มเติม
- สถานะบังคับใช้ / เลิกใช้
- ตัวบทที่เกี่ยวข้องกับกรณีจริง
- ประกาศหรือระเบียบปฏิบัติที่ออกภายหลัง
