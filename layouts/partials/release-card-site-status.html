เว็บไซต์นี้อยู่ในสถานะใด

เว็บไซต์ กฎหมายการศึกษา (Education Law Hub) เปิดใช้งานในระยะ Public Beta — เป็นการเปิดให้ผู้ใช้ทั่วไปเข้าถึงได้แล้ว ขณะที่ทีมยังคงทบทวนเนื้อหา แหล่งข้อมูล และฟีเจอร์อย่างต่อเนื่อง หมายความว่า:

  • เปิดใช้งานสาธารณะแล้ว — เข้าได้ผ่าน GitHub Pages โดยไม่ต้องล็อกอิน
  • ฟีเจอร์หลักพร้อมใช้งาน — คลังกฎหมาย ค้นหา แบบทดสอบ เส้นทางการเรียนรู้ คู่มือ และภาพรวมเว็บไซต์ จัดทำเพื่อการเรียนรู้เบื้องต้น
  • เนื้อหายังพัฒนาอยู่ — อาจมีการเพิ่มกฎหมาย ปรับสรุป แก้ลิงก์ หรืออัปเดต metadata โดยไม่ถือเป็น “ฉบับสมบูรณ์สุดท้าย”
  • รับข้อเสนอแนะจากผู้ใช้ — แจ้งข้อผิดพลาดหรือแหล่งที่ควรเพิ่มได้ผ่าน หน้าแจ้งข้อเสนอแนะ
  • ยังไม่ใช่เว็บไซต์ราชการ — ไม่ได้จัดทำหรือรับรองโดยหน่วยงานของรัฐ
  • ยังไม่ใช่บริการให้คำปรึกษาทางกฎหมาย — ไม่ตอบคำถามเฉพาะกรณีหรือข้อพิพาทส่วนบุคคน
  • ไม่รับประกันความครบถ้วน — คลังกฎหมายเป็นการคัดสรรและสรุป ไม่ได้รวบรวมทุกฉบับในประเทศ และไม่รับประกันว่าทุกหน้าเป็นฉบับล่าสุด
  • ควรตรวจแหล่งทางการก่อนใช้จริง — ก่อนอ้างอิงอย่างเป็นทางการ ต้องเปิดดูตัวบทฉบับทางการจากราชกิจจานุเบกษา กฤษฎีกา หรือหน่วยงานที่เกี่ยวข้อง
