## Desktop (Chrome หรือ Firefox ล่าสุด)

### หน้าแรก
- [ ] Hero, สถิติ, และ CTA แสดงถูกต้อง
- [ ] การ์ดหมวดหมู่กฎหมายลิงก์ไปคลังได้
- [ ] ส่วน「เรียนรู้ได้หลายรูปแบบ」ลิงก์ครบ (เส้นทาง สรุป คู่มือ แบบทดสอบ FAQ)
- [ ] Footer ลิงก์นโยบายและสถานะเวอร์ชัน

### คลังกฎหมาย
- [ ] `/laws/` แสดงรายการกฎหมาย
- [ ] ตัวกรอง `?cat=` ทำงาน
- [ ] หน้า `/laws/nea-2542/` (หรือฉบับอื่น) — metadata card, disclaimer, ปุ่มแหล่งทางการ
- [ ] ลิงก์「แจ้งข้อผิดพลาด」ไป `/feedback/` พร้อม query หน้า

### ค้นหา
- [ ] `/search/` โหลด Pagefind UI
- [ ] ค้นหาคำ เช่น “วิทยฐานะ” หรือ “การศึกษาแห่งชาติ” ได้ผลลัพธ์
- [ ] คลิกผลลัพธ์ไปหน้าที่ถูกต้อง

### Taxonomy
- [ ] `/categories/` หรือลิงก์หมวดจาก footer
- [ ] `/audiences/` หรือหน้ากลุ่มผู้ใช้
- [ ] หน้า term แสดงกฎหมายที่เกี่ยวข้อง

### แบบทดสอบ
- [ ] `/quiz/` เลือกหมวดได้
- [ ] ตอบคำถาม แสดงเฉลย
- [ ] ปุ่มเริ่มใหม่ทำงาน

### เส้นทางการเรียนรู้
- [ ] `/learning-paths/` แสดงการ์ด 6 เส้นทาง
- [ ] เปิด `/learning-paths/teachers/` — เนื้อหา ลำดับขั้น ลิงก์กฎหมาย

### คู่มือ
- [ ] `/guides/` แสดง 6 บทความ
- [ ] เปิดบทความหนึ่ง — taxonomy badges, related resources, disclaimer

### ภาพรวมและ feedback
- [ ] `/dashboard/` ตัวเลขและตารางแสดง
- [ ] `/feedback/` ช่องทางแจ้งและข้อความ privacy

### นโยบายและสถานะ release
- [ ] `/about/source-policy/`
- [ ] `/about/update-policy/`
- [ ] `/about/reliability/`
- [ ] `/about/public-beta/` — ข้อจำกัดชัดเจน ไม่ overclaim

## Mobile (จอแคบ ≤ 390px หรือ DevTools)

- [ ] เมนู hamburger เปิด/ปิดได้ ไม่บังเนื้อหา
- [ ] หน้าแรกอ่านได้ ไม่ต้อง scroll แนวนอน
- [ ] ตาราง dashboard scroll แนวนอนได้ (ถ้ามี)
- [ ] ปุ่ม CTA กดง่าย (ขนาดแตะเพียงพอ)
- [ ] ค้นหาและแบบทดสอบใช้งานได้

---

## SEO (ตัวอย่าง 2–3 หน้า)

- [ ] View Source — มี `<title>` และ `meta description` สมเหตุสมผล
- [ ] มี `link rel="canonical"`
- [ ] Open Graph `og:title`, `og:description`, `og:url` ไม่ว่าง
- [ ] ไม่มีคำอ้าง “ครบถ้วน” ใน meta description

---

## การเข้าถึงเบื้องต้น

- [ ] มี skip link「ข้ามไปยังเนื้อหาหลัก」
- [ ] หัวข้อเป็นลำดับชั้น (h1 → h2)
- [ ] ลิงก์ภายนอกที่สำคัญมี `rel="noopener"`
- [ ] สี contrast อ่านได้บนพื้นหลังหลัก

---

## ประสิทธิภาพเบื้องต้น

- [ ] หน้าแรกโหลดเร็วพอสมควร (static site)
- [ ] ไม่มี error ร้ายแรงใน Console บนหน้าหลัก
- [ ] รูปโลโก้/feature โหลดได้
