> **หมายเหตุ:** เว็บไซต์นี้จัดทำขึ้นเพื่อการศึกษา ค้นคว้า และการทำความเข้าใจกฎหมายการศึกษาเบื้องต้น ไม่ถือเป็นคำปรึกษากฎหมายเฉพาะกรณี

## วัตถุประสงค์ของนโยบายแหล่งข้อมูล

นโยบายนี้อธิบายว่าเว็บไซต์ **กฎหมายการศึกษา** ใช้แหล่งข้อมูลประเภทใด จัดลำดับความน่าเชื่อถืออย่างไร และผู้ใช้ควรอ้างอิงแหล่งใดเมื่อต้องการข้อมูลที่แม่นยำสำหรับการใช้งานจริง

เป้าหมายคือให้ครู ผู้บริหารสถานศึกษา เจ้าหน้าที่ด้านการศึกษา นักวิจัย และประชาชนทั่วไป เข้าใจว่าข้อมูลบนเว็บไซต์นี้มาจากไหน และควรตรวจสอบกับแหล่งทางการเมื่อใด

## แหล่งข้อมูลทางการที่ใช้เป็นหลัก

เว็บไซต์นี้ให้ความสำคัญกับแหล่งข้อมูลทางการและแหล่งที่หน่วยงานของรัฐเผยแพร่เป็นหลัก โดยเฉพาะ:

- **ราชกิจจานุเบกษา** — ตัวบทกฎหมาย ประกาศ ระเบียบ หรือข้อบังคับที่ประกาศใช้บังคับตามกระบวนการทางกฎหมาย
- **สำนักงานคณะกรรมการกฤษฎีกา (กฤษฎีกา)** — ฐานข้อมูลกฎหมายและคำวินิจฉัยที่เผยแพร่ทางการ
- **เว็บไซต์หน่วยงานที่เกี่ยวข้องกับการศึกษา** — เช่น กระทรวงศึกษาธิการ สพฐ. สอศ. สำนักงาน ก.ค.ศ. คุรุสภา กสศ. กยศ. และหน่วยงานอื่นที่มีอำนาจหน้าที่ตามกฎหมาย
- **รัฐสภาไทย** — ร่างกฎหมาย บันทึกการพิจารณา หรือข้อมูลที่เผยแพร่ทางการเกี่ยวกับกฎหมายที่อยู่ระหว่างดำเนินการ

เมื่อมีลิงก์ไปยังแหล่งทางการในหน้ากฎหมายแต่ละฉบับ ให้ถือว่าเป็นแหล่งอ้างอิงหลักที่ผู้ใช้ควรเปิดอ่านเพิ่มเติม

## ลำดับความน่าเชื่อถือของแหล่งข้อมูล

เมื่อข้อมูลจากหลายแหล่งไม่ตรงกัน ให้ใช้หลักการจัดลำดับดังนี้:

1. **ตัวบทกฎหมายฉบับเต็มที่ประกาศในราชกิจจานุเบกษา** (หรือฉบับที่ยังมีผลบังคับใช้ตามที่ระบุในแหล่งทางการ)
2. **ประกาศ ระเบียบ ข้อบังคับ หรือเอกสารทางการของหน่วยงาน** ที่ออกตามอำนาจตามกฎหมาย
3. **คำชี้แจงหรือคู่มือที่หน่วยงานเผยแพร่ทางการ** บนเว็บไซต์หรือช่องทางอย่างเป็นทางการของหน่วยงานนั้น
4. **สรุปสาระสำคัญบนเว็บไซต์นี้** — จัดทำเพื่อช่วยให้เข้าใจภาพรวม ไม่แทนตัวบทกฎหมาย
5. **ข่าว บทความ หรือคำอธิบายจากบุคคลภายนอก** — ใช้เป็นแนวทางเบื้องต้นได้ แต่ไม่ควรอ้างอิงแทนแหล่งทางการ

**หลักสำคัญ:** ตัวบทกฎหมายและเอกสารทางการของหน่วยงานควรได้รับความสำคัญกว่าสรุป ข่าว หรือคำอธิบายที่ไม่เป็นทางการเสมอ

## การใช้ข้อมูลจากเว็บไซต์หน่วยงาน

เว็บไซต์หน่วยงานของรัฐอาจมีการปรับปรุงหน้าเว็บ ย้ายที่อยู่ลิงก์ หรือเผยแพร่ฉบับแก้ไขเพิ่มเติม ดังนั้น:

- ลิงก์ที่แสดงบนเว็บไซต์นี้มีไว้เพื่อช่วยนำทางไปยังแหล่งทางการ แต่ไม่รับประกันว่าจะเปิดได้หรือเป็นฉบับล่าสุดเสมอไป
- ข้อความที่คัดลอกหรือสรุปจากเว็บไซต์หน่วยงาน อาจไม่ครอบคลุมรายละเอียดทั้งหมดในต้นฉบับ
- หากพบว่าข้อมูลบนหน้าเว็บหน่วยงานเปลี่ยนแปลง ควรยึดข้อมูลจากหน่วยงานนั้นเป็นหลัก

## การใช้สรุปหรือคำอธิบายกฎหมาย

เนื้อหาสรุปบนเว็บไซต์นี้มีจุดประสงค์เพื่อ:

- ช่วยให้ผู้ที่ไม่ใช่นักกฎหมายเข้าใจโครงสร้างและประเด็นสำคัญของกฎหมายการศึกษา
- จัดหมวดหมู่และเชื่อมโยงกฎหมายที่เกี่ยวข้องให้ค้นหาได้ง่าย
- สนับสนุนการเรียนรู้ผ่านคำถาม แบบทดสอบ และแหล่งอ่านเพิ่มเติม

สรุปอาจละรายละเอียด เงื่อนไข หรือข้อยกเว้นที่มีในกฎหมายฉบับเต็ม จึงไม่ควรนำไปใช้แทนการอ่านตัวบทกฎหมายหรือการปรึกษาผู้เชี่ยวชาญเมื่อมีผลทางกฎหมายจริง

## สิ่งที่เว็บไซต์นี้ไม่ทำ

เว็บไซต์นี้ **ไม่**:

- อ้างว่ารวบรวมกฎหมายการศึกษาทุกฉบับในประเทศได้ครบถ้วน
- แทนที่การตีความกฎหมายเฉพาะกรณีหรือการให้คำปรึกษาทางกฎหมาย
- รับประกันว่าข้อมูลทุกหน้าสะท้อนการแก้ไขเพิ่มเติมล่าสุดทันทีที่มีการประกาศใช้
- สร้างหรือแก้ไขตัวบทกฎหมาย — มีเพียงการสรุปและเชื่อมโยงไปยังแหล่งทางการ

## วิธีแจ้งข้อมูลผิดพลาดหรือแหล่งข้อมูลที่ควรแก้ไข

หากพบว่าลิงก์เสีย ชื่อกฎหมายไม่ตรงกับแหล่งทางการ หรือสรุปอาจทำให้เข้าใจคลาดเคลื่อน สามารถแจ้งได้ดังนี้:

- ใช้หน้า **[แจ้งข้อผิดพลาดและเสนอแนะ](/feedback/)** — เลือกแบบฟอร์มหรือ GitHub Issues
- เปิด **Issue** ที่ [GitHub repository ของโครงการ](https://github.com/burapatis/website-edulaw/issues/new/choose) พร้อมระบุชื่อกฎหมาย หน้าที่พบปัญหา และแหล่งทางการที่ควรอ้างอิง (ถ้ามี)

การแจ้งที่มีแหล่งทางการประกอบจะช่วยให้ตรวจสอบและปรับปรุงได้รวดเร็วขึ้น
