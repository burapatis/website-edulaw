<h2 id="available-features">สิ่งที่ใช้งานได้ในระยะแรก</h2>
<ul class="release-card__list release-card__list--features">
<li><strong>คลังกฎหมาย</strong> — สรุปกฎหมายการศึกษาที่คัดสรร พร้อม metadata และลิงก์แหล่งทางการ · <a href="/laws/">ไปที่คลัง</a></li>
<li><strong>ค้นหา</strong> — ค้นหาในเนื้อหาเว็บไซต์ · <a href="/search/">ไปที่หน้าค้นหา</a></li>
<li><strong>สำรวจตามหมวดหมู่</strong> — หมวด หน่วยงาน กลุ่มผู้ใช้ ประเภทกฎหมาย · <a href="/laws/">สำรวจคลัง</a></li>
<li><strong>แบบทดสอบ</strong> — ทบทวนความเข้าใจเบื้องต้น (ไม่ใช่การวินิจฉัยข้อกฎหมาย) · <a href="/quiz/">ไปที่แบบทดสอบ</a></li>
<li><strong>เส้นทางการเรียนรู้</strong> — ลำดับการอ่านตามบทบาท · <a href="/learning-paths/">เลือกเส้นทาง</a></li>
<li><strong>คู่มือและบทความอธิบาย</strong> — ภาพรวมประเด็นสำคัญ · <a href="/guides/">อ่านคู่มือ</a></li>
<li><strong>ภาพรวมเว็บไซต์</strong> — สถิติและสถานะ metadata เบื้องต้น · <a href="/dashboard/">ดูภาพรวม</a></li>
<li><strong>แจ้งข้อผิดพลาด/เสนอแนะ</strong> — ช่องทางรับ feedback · <a href="/feedback/">แจ้งข้อเสนอแนะ</a></li>
<li><strong>นโยบายและความน่าเชื่อถือ</strong> — แหล่งข้อมูล การอัปเดต ข้อจำกัด · <a href="/about/source-policy/">นโยบายแหล่งข้อมูล</a></li>
</ul>
</div>
<div class="release-card release-card--warning">
<h2 id="limitations">ข้อจำกัดสำคัญ</h2>
<ul class="release-card__list">
<li><strong>ไม่ใช่</strong> เว็บไซต์ของรัฐหรือหน่วยงานราชการ — เป็นโครงการจัดทำเพื่อการศึกษาและค้นคว้า</li>
<li><strong>ไม่ใช่</strong> บริการให้คำปรึกษาทางกฎหมายเฉพาะกรณี — หากมีผลทางกฎหมายจริง ควรปรึกษาผู้เชี่ยวชาญ</li>
<li><strong>ไม่รับประกัน</strong> ว่ารวบรวมกฎหมายการศึกษาทั้งหมดในประเทศ — มีเฉพาะฉบับที่คัดสรรในคลัง</li>
<li><strong>ไม่รับประกัน</strong> ว่าทุกหน้าเป็นฉบับล่าสุดหรือสรุปถูกต้องครบทุกกรณี — สถานะกฎหมายอาจเปลี่ยนหลังวันที่ตรวจสอบ</li>
<li>ลิงก์แหล่งทางการบางรายการอาจเสียหรือย้ายที่อยู่ — แจ้งได้ที่หน้า feedback</li>
<li>แบบทดสอบและคู่มือจัดทำเพื่อการศึกษา — ไม่ใช่มาตรฐานทางวิชาชีพหรือคำตัดสินทางกฎหมาย</li>
</ul>
</div>
<div class="release-card">
<h2 id="safe-use">วิธีใช้ข้อมูลอย่างปลอดภัย</h2>
<ol class="release-card__steps">
<li>อ่านสรุปบนเว็บไซต์เพื่อเข้าใจ<strong>ภาพรวม</strong> — ไม่ใช่ขั้นตอนสุดท้าย</li>
<li>เปิด <strong>แหล่งข้อมูลทางการ</strong> จากหน้ากฎหมาย (ปุ่มหรือ <code>source_url</code> ใน metadata card)</li>
<li>ตรวจ<strong>ชื่อเรื่อง ปี พ.ศ. สถานะ</strong> และฉบับแก้ไขเพิ่มเติมในแหล่งทางการ</li>
<li>อ่าน <a href="/guides/how-to-use-edulaw-safely/">คู่มือ: วิธีใช้เว็บไซต์อย่างปลอดภัย</a></li>
<li>หากใช้ในงานทางการ เอกสารราชการ หรือข้อพิพาท — ปรึกษาผู้เชี่ยวชาญหรือหน่วยงานที่เกี่ยวข้อง</li>
</ol>
</div>
<div class="release-card">
<h2 id="report-feedback">วิธีแจ้งข้อผิดพลาดหรือเสนอแนะ</h2>
<p>การมีส่วนร่วมช่วยให้เว็บไซต์น่าเชื่อถือขึ้น — โปรด<strong>อย่าส่งข้อมูลส่วนบุคคลที่อ่อนไหว</strong> (เลขประชาชน รายละเอียดคดี ฯลฯ)</p>
<ul class="release-card__list">
<li><a href="/feedback/">แจ้งข้อผิดพลาดและเสนอแนะ</a> — ช่องทางสำหรับผู้ใช้ทั่วไป</li>
<li><a href="https://github.com/burapatis/website-edulaw/issues/new/choose">GitHub Issues</a> — สำหรับผู้ใช้ทางเทคนิค (ข้อมูลสาธารณะ)</li>
</ul>
<p class="release-card__note">ข้อเสนอที่แนบ<strong>ลิงก์แหล่งทางการ</strong>จะช่วยให้ตรวจสอบและปรับปรุงได้เร็วขึ้น — การเสนอไม่เท่ากับการรับแก้ไขทุกเรื่อง</p>
</div>

<h2 id="roadmap">แผนการพัฒนาต่อ</h2>
<p>ทิศทางโดยย่อ — <strong>ไม่มีกำหนดเวลาแน่นอน</strong> และอาจปรับตามทรัพยากรและการตรวจแหล่งทางการ</p>
<ul class="release-card__list">
<li><strong>v1.1</strong> — ขยายคลังกฎหมายหลังตรวจแหล่งทางการ</li>

<li><strong>v1.4</strong> — ปรับปรุงภาพรวมเว็บไซต์และรายงาน</li>

</ul>
<p class="release-card__note">รายละเอียดแผนพัฒนาอยู่ใน <code>docs/ROADMAP.md</code> ใน repository โปรเจกต์ — หน้านี้จะอัปเดตเมื่อมีเวอร์ชันใหม่</p>
</div>
