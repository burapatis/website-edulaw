# บันทึกการตรวจสอบแหล่งข้อมูล (Source Review Log)

ไฟล์นี้ใช้บันทึกการตรวจสอบแหล่งทางการ **ด้วยมือ** — สคริปต์ `npm run check:content` ไม่แทนการตรวจนี้

## วิธีใช้

1. เมื่อเพิ่มหรือทบทวนกฎหมาย ให้เพิ่มแถวในตารางด้านล่าง
2. ระบุ URL หรือชื่อแหล่งที่เปิดอ่านจริง
3. สรุปผลสั้น ๆ (เช่น “ตรงกับฉบับ”, “ลิงก์เสีย”, “ต้องอัปเดตสรุป”)

## ตารางบันทึก

| วันที่ตรวจสอบ | ไฟล์/หน้า | เรื่องที่ตรวจสอบ | แหล่งข้อมูลทางการที่ใช้ | ผลการตรวจสอบ | ผู้ตรวจ/หมายเหตุ |
|---------------|-----------|------------------|-------------------------|---------------|------------------|
| _(ตัวอย่าง — ลบแถวนี้เมื่อบันทึกจริง)_ | `data/laws.json` → `nea-2542` | พ.ร.บ.การศึกษาแห่งชาติ | _(ใส่ URL จริง)_ | _(รอตรวจ)_ | _(ชื่อ)_ |

---

_ยังไม่มีบันทึกการตรวจสอบใน repo นี้ — เพิ่มแถวแรกเมื่อมีการตรวจแหล่งจริง_

## รอบตรวจแหล่งข้อมูล ขั้นที่ 16

รอบนี้มุ่งยืนยันแหล่งข้อมูลทางการของหน้ากฎหมายสำคัญ (P1/P2) ดูรายละเอียดและลำดับงานที่:

- `docs/SOURCE_VERIFICATION_AUDIT.md` (สร้างด้วย `npm run audit:sources`)
- `docs/PRIORITY_LAW_SOURCE_CHECKLIST.md`
- `docs/SOURCE_VERIFICATION_WORKSHEET.md` (วิธีตรวจและกรอกข้อมูล)

กรอกแถวด้านล่างเมื่อตรวจแต่ละฉบับเสร็จ — **อย่ากรอกล่วงหน้าหรือใส่ผลที่ยังไม่ได้ตรวจจริง**
